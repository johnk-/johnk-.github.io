Build System Discuss rebol-make and other build systems. Semantics While the "Internals" discusses the bits and bytes of how things are implemented, this category is for discussing things design questions of the language... like what do [] should return. Foreign Inspiration Take inspiration from other languages to create the same functionality in rebol. Datatypes The datatypes category is for the discussion of the specific properties of datatypes themselves. Sample topics might include: Modules Module Development Focus GitHub List/discuss the common commands that are used when working at a simple level with git and the sources. Debugger It's best to walk before you run! Binding "Binding" refers to the concept of an invisible pointer that is annotated onto ANY-WORD! types, which connects them to an object where that word is defined (or more specifically, an ANY-CONTEXT!)
