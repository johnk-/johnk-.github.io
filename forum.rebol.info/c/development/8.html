Semantics While the "Internals" discusses the bits and bytes of how things are implemented, this category is for discussing things design questions of the language... like what do [] should return. Modules Rebol2 was developed for small scripts, and didn't have any concept of "module isolation". Optimization This is a category for discussing performance and optimization ideas. Foreign Inspiration Take inspiration from other languages to create the same functionality in rebol.
