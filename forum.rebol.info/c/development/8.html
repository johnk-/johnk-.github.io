Optimization This is a category for discussing performance and optimization ideas. GitHub List/discuss the common commands that are used when working at a simple level with git and the sources. Internals Internals of the Rebol language Web There are currently two ways in which Ren-C is built to WebAssembly: Language Bridging Interfacing Rebol with other languages discussed here. Debugger It's best to walk before you run! Semantics While the "Internals" discusses the bits and bytes of how things are implemented, this category is for discussing things design questions of the language... like what do [] should return. Datatypes The datatypes category is for the discussion of the specific properties of datatypes themselves.
