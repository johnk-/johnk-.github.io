Ports To discuss the port model and also the implementation of ports with respect to networking. Optimization This is a category for discussing performance and optimization ideas. Semantics While the "Internals" discusses the bits and bytes of how things are implemented, this category is for discussing things design questions of the language... like what do [] should return. Errors For discussion of error handling and error mechanisms. Build System Discuss rebol-make and other build systems. Functions Discussions of Functions and Generators, e.g. how things like RETURN work... or the mechanics of generators and composers like ADAPT, SPECIALIZE, ENCLOSE, HIJACK, etc. Datatypes The datatypes category is for the discussion of the specific properties of datatypes themselves. Sample topics might include: Testing Discussion of Tests, Testing Methodology and Continuous Integration (e.g. Travis-CI). Foreign Inspiration Take inspiration from other languages to create the same functionality in rebol. Modules Module Development Focus GitHub List/discuss the common commands that are used when working at a simple level with git and the sources. Binding "Binding" refers to the concept of an invisible pointer that is annotated onto ANY-WORD! types, which connects them to an object where that word is defined (or more specifically, an ANY-CONTEXT!) Debugger It's best to walk before you run! Language Bridging Interfacing Rebol with other languages discussed here. Naming Suggestions on naming new functions, and guidance on choosing names.
