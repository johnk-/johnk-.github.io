Rebmu Rebmu is a dialect for solving Code Golf challenges, where the goal is to express the source code for solving a puzzle in the fewest characters of source code possible. See the README here: Cryptography Discussion of things related to cryptography, which includes low level primitives (ciphers, hashes, and key exchange) as well as higher-level protocols such as TLS. Parsing Discussion focusing on parsing data. Redbol Issues regarding compatibility and emulation of Rebol2 and Red. General Tools Discuss various utilities written in Rebol here that aren't system tools eg. rebolbot, twitter Dialects/DSLs Discussion focused on the creation of dialects and DSLs using Rebol. Rebmu talk is appropriate here. Help System This section is for discussing commands like HELP and SOURCE. System Tools Discussion focusing on creating useful system utilities (e.g. replacement for grep, sed, awk, etc.) Web Topics related to the WebAssembly build and in-browser applications. Console Specific development relating to the console, and driving the same.
