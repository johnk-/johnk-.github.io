jjsullivan5196 May 22, 2024, 7:36pm 4

Should probably mention clojure spec has combinators for validation rules and conform as a not-so-well documented protocol for making productions based on those rules. There's some very simple productions for doing things like parsing argument lists and other sequential messages. You can get pretty fancy with it if you know what you're doing, but it probably could use some more inspiration from PARSE.
