(Not everyone has seen this, e.g. @bradrn probably. And it predates the forum by quite a long time. So I thought I'd post it.)

Around 2015 DocKimbel asked if I would look into the idea of designing a C/C++ API for Red. Since Red was incomplete at the time, I built it as an abstraction layer over either Rebol or Red... and used Rebol for my tests and demos.

Once I had the API I thought it would be neat to do something with it. So I built a GUI console in Qt.

Looking back at it, the demo was actually pretty impressive. Qt is pretty good for writing this kind of thing, so I was able to put it together--both the API and the console--in about a month:

(As I mention in the video description, I type very fast and accurately...but not in this video as I was dealing with the cat who was getting in the way of me and the microphone/keyboard.)
